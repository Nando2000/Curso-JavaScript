Eventos DOM

Os Eventos no DOM, nada mas são que ações que correm dentros das tags, como por exemplo quando colocamos o mouse dentro de uma div, pode ocorer diversas ações como:

  1. mouseenter = Quando o mouse entra na tag
  2. mousemove = Quando é movido o mouse
  3. click = Click
  4. Entre outros...
E para poder tabalhar com esses eventos precisamos ultilizar as função dentro do JS.
