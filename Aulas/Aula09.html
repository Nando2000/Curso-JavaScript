Elemento de seleção


Podemos selecionar qualquer elemento do html com seguinte comando: window.document.getElementsByTagName('p')[0] colocamos indice para especificar qual parágrafo queremos selcionar, pois sem ele selecionaria todos os elementos específicados('p') no html. Podemos também selcionar tags pelo seu id, classe, nome ou por um seletor:

  1. ID: window.document.getElementsByID('nomeid')
  2. CLASE: window.document.getElementsByClasseName('nomeclase')
  3. Nome: window.document.getElementsByName('nome')


Obs: A função inerText busca somente o texto, agora caso se quisermos ter o texto com a formatção usamos innerHTML
